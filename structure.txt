📊 МОДЕЛИ ДАННЫХ:
═══════════════════════════════════════════════════════════════

1. Workshop (Цех)
   - name, code, description

2. Site (Площадка)
   - name, workshop, description

3. EquipmentType (Тип оборудования)
   - name, description

4. Equipment (Оборудование) ⭐ Главная модель
   - name, inventory_number, equipment_type, site
   - manufacturer, model, serial_number, year
   - status, notes, created_at, updated_at

5. EquipmentCharacteristic (Характеристика)
   - equipment, name, value, unit

6. EquipmentDocument (Документ)
   - equipment, title, file, uploaded_at
